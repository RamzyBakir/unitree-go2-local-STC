Overview

Fully local.
No cloud required.

A fully local, self-hosted voice control system for the Unitree Go2. It runs an embedded Whisper STT model, matches spoken words to a predefined command table, and sends control signals to the robot over DDS (CycloneDDS) — no internet required after setup.

Get Started

One command away
from voice control.

Clone, install, configure, run. The dashboard starts in seconds.

# 1. Clone the fork
$ git clone https://github.com/RamzyBakir/unitree-go2-local-STC.git
$ cd unitree-go2-voice-control

# 2. Install base SDK + voice deps
$ pip3 install -e .
$ pip3 install -r voice_control/requirements.txt

# 3. Configure .env
$ cp .env.example .env
$ nano .env # Set GO2_NETWORK_INTERFACE, GO2_AUTH_PASSWORD, etc.

# 4. Launch
$ python3 run.py

# Open http://localhost:8000 in your browser
